StExBar command options

Separator

If enabled, the command is made a separator line.

Name

The name of the Command, shown as a tooltip on the toolbar button. Each command must have a unique name.

Icon

Path to an icon to be used for the command. If left empty, StExBar tries to find the icon itself.

Command line

The command line to execute. Environment variables are expanded properly. The following placeholders are available:
  • %selpaths: will be replaced with the paths of the selected items, separated by a space
  • %sel*paths: will be replaced with the paths of the selected items, separated by "*" char
  • %selafile: replaced with the path to a file containing the paths in ANSI of the selected items separated by a newline char
  • %selufile: replaced with the path to a file containing the paths in Unicode of the selected items separated by a newline char
  • %selnames: replaced with the names of the selected items
  • %curdir: replaced with the path of the currently shown directory
  • %cmdtext: replaced with the content of the edit box

Hotkey

The hotkey to use for that particular command. The hotkey will only work in Explorer.
