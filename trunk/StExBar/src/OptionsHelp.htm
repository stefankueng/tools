StExBar Options
Show button text on toolbar
if enabled, the command text appears next to each button on the toolbar

Copy UNC paths
if enabled, the command "Copy path" will copy the UNC path instead of the path of the mapped network drive. For example, instead of N:\folder\myfile.txt the path \\server\folder\myfile.txt is copied to the clipboard. This is useful if other users don't have the same mapping of network folders and you want to pass those paths around.

Hide edit box
if enabled, the edit box usually shown at the very right of the toolbar is not shown.
Use selector instead of cmd replacement
if this is enabled, pressing ENTER in the edit box will select all files matching the string in the edit box. If disabled, then pressing ENTER will execute the string as if it were entered in the console.
